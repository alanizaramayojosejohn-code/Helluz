Gestión de Usuarios

Administra los usuarios del sistema

Instrucciones para crear usuarios:

  1. Ir a Firebase Console → Authentication → Users → Add User
  2. Crear usuario con email y contraseña
  3. Copiar el UID del usuario creado
  4. Hacer clic en "Nuevo Usuario" y completar los datos con el UID
